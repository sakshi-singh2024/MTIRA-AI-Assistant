# 🤖 MTIRA - AI Voice Assistant

MTIRA is an interactive AI-powered voice assistant with a 3D avatar interface.

It supports voice commands, speech synthesis, lip-sync animation, and YouTube interaction.

---

## 🚀 Features

- 🎤 Wake word detection ("Rukmani")
- 🗣 Speech Recognition (Web Speech API)
- 🔊 Female Voice Output (Speech Synthesis)
- 👩‍💻 Avatar with Lip Sync
- 📺 YouTube Search & Playback
- 💬 Offline + AI-based Responses
- ⚡ Event-driven architecture

---

## 🛠 Tech Stack

- HTML, CSS, JavaScript
- Web Speech API
- Speech Synthesis API
- Iframe-based Avatar System
- Async API Handling (Fetch)

---

## 🧠 Key Concepts Used

- Event-driven programming
- Asynchronous operations (async/await)
- Voice processing pipeline
- UI + AI system integration
